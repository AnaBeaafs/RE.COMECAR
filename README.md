# RE.COMEÇAR  
**Auxiliando o público mais velho a se reconectarem com o mundo digital**

O **RE.COMEÇAR** é um projeto desenvolvido para apoiar adultos acima de 40 anos no processo de aprendizagem de ferramentas tecnológicas essenciais para o dia a dia.  
O site apresenta conteúdos simples, objetivos e acessíveis sobre ferramentas como **Word**, **Canva**, **edição de vídeo mobile**, uso de **IA (como o ChatGPT)** e muito mais — sempre com foco em autonomia, clareza e acessibilidade.

## Justificativa e Objetivo

O projeto **RE.COMECAR** surgiu de uma vivência **real** de uma das integrantes da equipe, que constantemente recebia mensagens de uma colega pedindo ajuda para realizar tarefas simples no computador, como **abrir o Word**, criar um post no **Canva** ou entender como funcionavam **ferramentas digitais simples**.

A persona que inspirtou nosso projeto é **Paulinha**, de **42 anos**, sempre foi dedicada e trabalhou
como monitora em uma escola. Quando surgiu a oportunidade de assumir o cargo de **criadora
de mídias** no mesmo colégio, ela se viu diante de um **grande desafio:** dominar ferramentas
tecnológicas que **nunca havia utilizado.**

**Sem base técnica** e com **insegurança** em relação ao mundo digital, ela passou a depender
**constantemente** de ajuda para realizar atividades básicas — uma situação que representa a
realidade de **muitas** pessoas na mesma **faixa etária.**

**Essa experiência despertou o grupo para um problema social relevante:**
A falta de inclusão digital e a dificuldade de reinserção profissional de adultos maduros
em um mercado cada vez **mais tecnológico.**

A partir dessa vivência concreta, nasceu a motivação do **RE.COMEÇAR**: criar um portal **acessível,
humano** e **acolhedor** que ofereça orientação, capacitação e apoio para pessoas que desejam
se **reinventar** profissionalmente, mas não sabem por onde começar.

A **ideia central** é mostrar que o futuro do trabalho é **inclusivo** e **acessível para todos**, e que
aprender tecnologia não depende de idade, mas de **oportunidade** e **incentivo.**
Assim, o site **RE.COMEÇAR** une educação, tecnologia e empatia, permitindo que pessoas como Paulinha descubram novas possibilidades profissionais e se sintam **preparadas** para o
mercado digital.

## Funcionalidades
- Guia introdutório sobre ferramentas digitais populares  
- Interface acessível, responsiva e intuitiva  
- Ajuste de tamanho de fonte (16px ↔ 20px)  
- Ícones ilustrativos e dinâmicos com Lucide React  
- Suporte total a acessibilidade com **VLibras** (tradução para LIBRAS)

---

## Tecnologias Utilizadas

### **1. React + Vite**
- **React 18.2.0** para criação de componentes reutilizáveis  
- **Vite** para build rápido e HMR  
- Uso de **React Hooks** (`useState`, `useEffect`) para estados e lógica

### **2. Bootstrap 5**
- Sistema de grid responsivo  
- Componentes prontos como botões, modais e collapses  
- Utilização de utilitários e transições suaves em hover

### **3. Lucide React**
- Biblioteca moderna de ícones SVG  
- Ícones como `RefreshCw`, `Mail`, `Rocket`, `Sparkles` etc.  
- Ícones acessíveis com `aria-label`

### **4. CSS Modules**
- Estilização modular e com escopo local  
- Evita conflitos de classe  
- Suporte a animações com `@keyframes`

### **5. VLibras (Acessibilidade em LIBRAS)**
Integração via script:

```html
<script src="https://vlibras.gov.br/app/vlibras-plugin.js"></script> 
```

O **RE.COMEÇAR** nasce como uma iniciativa de inclusão digital, oferecendo recursos simples, acessíveis e intuitivos para apoiar pessoas **acima dos 40 anos** na jornada de aprendizado tecnológico.
